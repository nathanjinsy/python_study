# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a structured Python learning plan (`python_study.md`) for a 9-year-old child learning to code from scratch. It is a single Markdown document — there is no runnable code, build system, or tests.

## Content Structure

`python_study.md` is organized as a 6-stage curriculum:

1. **Basics** — print, variables, data types, input, strings
2. **If / Else** — conditionals, comparison/logical operators, `random` module
3. **Loops** — `for`, `while`, `range()`, `break`, `continue`
4. **Functions** — `def`, parameters, `return`, scope
5. **Lists** — indexing, `.append()`, `.remove()`, `.sort()`, iteration
6. **Final Project** — choose a game (Snake & Ladder, Blackjack, Hangman, etc.)

Each stage includes: prerequisite knowledge recap, concept checklist, mini projects, and a freeform notes section. There is also a session log table, milestone rewards tracker, teaching tips for the parent, helpful resource links, and a bonus stage covering dictionaries, file I/O, Turtle graphics, pygame, APIs, and classes.

## Editing Guidelines

- Checkboxes use `- [ ]` (unchecked) and `- [x]` (checked) Markdown syntax; status emoji (`⬜`, `🔄`, `✅`) are used in the progress table.
- Preserve the teaching tone: friendly, encouraging, aimed at a child with a parent co-learning alongside.
- Keep concept explanations concrete with short code examples in inline code blocks.
- The "Tips for Dad" section is for the parent-teacher, not the child — keep that voice distinct.
